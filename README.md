# Does money buy happiness? — an interactive retelling

A mobile-first, scrollytelling walk through **Killingsworth, Kahneman & Mellers (2023),
"Income and emotional well-being: A conflict resolved"** (PNAS).

The reader plays with a reconstruction of the paper's Figure 2 / Table 1 (drag a "baseline mood"
dial and watch the income→happiness curve change shape), then reads the full open-access PDF inline.

Static site — no build step. Charts: Canvas 2D. PDF: pdf.js.
